# Roadmap - Red CheoSys

**Última actualización:** 2026-06-28
**Estado:** Fase 1 - MVP ✅ COMPLETADO

## 🎯 Visión
Motor de búsqueda humanitario descentralizado, ultra-ligero, open source.
Tipo "Google de los 90s con tecnología de 2026".

## 📊 Fases

### Fase 1: Core (Semana 1-2) ✅ COMPLETADO
- [x] Next.js 14 + TypeScript + Tailwind
- [x] Formulario de recolección de datos
- [x] Meilisearch instalado y configurado
- [x] Búsqueda full-text instantánea (<100ms)
- [x] Home minimalista tipo Google
- [x] Sistema de imágenes con Sharp.js
- [x] Geolocalización con OpenStreetMap
- [x] Búsqueda por cercanía

### Fase 2: Optimización (Semana 3) 🔄 EN PROGRESO
- [x] Upload de imágenes con optimización automática
- [x] Conversión a WebP (800x800px, calidad 70%)
- [x] Almacenamiento organizado por año/mes
- [ ] PWA + Service Worker
- [ ] Cache strategies
- [ ] Compresión Brotli en Nginx

### Fase 3: Comunidad (Semana 4)
- [ ] Sistema de votos/confianza
- [ ] Reportar contenido
- [ ] RSS feeds
- [ ] API pública documentada
- [ ] Moderación automática

### Fase 4: Escalamiento (Semana 5-6)
- [ ] Donaciones cripto (USDT, Bitcoin)
- [ ] Analytics simple
- [ ] Búsqueda por voz
- [ ] Búsqueda por SMS
- [ ] Multi-idioma (i18n)

## 📈 Métricas Actuales
- ✅ Tiempo de búsqueda: <10ms (Meilisearch)
- ✅ First Contentful Paint: <1s
- ✅ Bundle size: ~96KB (optimizado)
- ✅ Imágenes: WebP optimizado (50-100KB promedio)
- ✅ Lighthouse score: >95
